# Google_form_automation
This is google form automation build using python selenuim. It asks for how many fillups do user need and fills form as many times given by the user.

How to run this
1. Install python latest version
2. Clone this repo
3. Open in code
4. Install reqirements.py (pip install -r requirements.py)
5. Rub the python script (python googl_form_automation.py)
